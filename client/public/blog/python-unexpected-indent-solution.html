Python IndentationError: unexpected indent — explained and fixed

Python treats whitespace as syntax. Mixing tabs and spaces or adding a stray indent is all it takes to trigger IndentationError: unexpected indent. Here is a short, actionable guide to fix it and keep your files clean.

Root causes

  • Tabs and spaces mixed in the same block.
  • An extra space before a function, loop, or conditional.
  • Copying code from the web with non-breaking spaces.

Before and after

# Broken: stray indent before print
def greet(name):
        message = f"Hello {name}"
    print(message)
# Fixed: consistent 4-space indents
def greet(name):
    message = f"Hello {name}"
    print(message)

Quick fix checklist

  1. Enable "soft tabs" (spaces) in your editor. The online compiler uses 4 spaces by default.
  2. Select all and convert tabs to spaces.
  3. Align blocks under def, if, for, and while.
  4. Re-run the code and confirm the error disappears.
